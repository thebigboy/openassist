# OpenAssist 助手 - Chrome 插件

这是一个基于 Chrome Side Panel API 开发的 AI 助手插件。

## 主要功能
- **侧边栏对话**：点击插件图标或通过右键菜单即可唤起侧边栏。
- **右键解释**：选中网页文字后，右键选择“使用 OpenAssist 解释选中文字”，插件会自动获取内容并准备回答。
- **现代化 UI**：简洁、响应式的对话界面。

## 安装步骤
1. 下载并解压 `openassist-extension.zip`。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 在右上角开启“开发者模式”。
4. 点击左上角的“加载已解压的扩展程序”。
5. 选择解压后的文件夹即可完成安装。

## 开发者说明
- 本项目采用 **Manifest V3** 标准。
- AI 回复目前为模拟逻辑，开发者可根据需要在 `sidepanel/script.js` 中接入真实的 API 接口。
- 图标由 AI 生成，包含 16x16, 48x48, 128x128 三种尺寸。
